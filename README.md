# 🐾 PawMart Florida – Premium Pet Essentials
**Live Experience:** [https://mrprotik.github.io/pawmart-florida](https://mrprotik.github.io/pawmart-florida)

PawMart Florida is an ultra-vibrant, luxurious multi-page e-commerce demo designed for the discerning pet owner in the Sunshine State. This project showcases a high-end "boutique" digital experience, blending sophisticated Florida-inspired aesthetics with a seamless, modular functional core.

---

## 💎 The Experience
The website is built on a custom **Cream, Navy, and Gold (#D4AF87)** palette, evoking the prestige of Palm Beach and Miami’s high-end retail districts. 

### Key Features:
* **Curated Pet Suites:** Dedicated, immersive categories for Dogs, Cats, Fish, Birds, and Small Pets.
* **Executive Admin Portal:** A private "hidden" dashboard (admin/demo123) for managing inventory and orders.
* **Real-Time Luxury Cart:** A persistent Vanilla JS cart system that tracks selections across all 9 pages.
* **Responsive Fluidity:** Mobile-first design ensuring the "8k" luxury feel on iPhones, tablets, and desktops.
* **Concierge Services:** A dedicated contact portal for bespoke pet audits and professional inquiries.

---

## 🛠️ Tech Stack
* **Architecture:** Semantic HTML5 (9 separate linked modules).
* **Styling:** Tailwind CSS via CDN with custom luxury extensions.
* **Interactions:** Vanilla JavaScript (No heavy frameworks, ensuring ultra-fast load times).
* **Typography:** Playfair Display (Serif) & Inter (Sans-Serif) via Google Fonts.
* **Assets:** AI-generated high-fidelity commercial pet photography.
